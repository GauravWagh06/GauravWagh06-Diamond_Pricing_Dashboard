# Diamond Pricing Analysis Dashboard
## Overview

This project presents an interactive Power BI dashboard designed to analyze the relationship between diamond quality characteristics and pricing.

The dashboard helps users understand how factors such as carat, cut, color, clarity, and dimensions influence the value of diamonds.

---

## Tools Used

* Power BI
* CSV Dataset
* Data Cleaning
* Data Visualization
* DAX Measures

---

## Business Objectives

* Analyze the relationship between carat weight and price
* Compare average prices by cut, color, and clarity
* Identify premium diamond categories
* Detect pricing and dimension outliers
* Explore high-value quality combinations

---

## Dashboard Features

* KPI Cards
* Interactive Filters
* Price vs Carat Analysis
* Price by Cut Quality
* Price by Color Grade
* Price by Clarity Level
* Dimension Analysis
* Dynamic Visualizations
